# Contributing

This repository is designed as a practical step-by-step user guide for Windows, Active Directory and IT Support practice.

## Guidelines

When adding or updating content:

- Keep instructions clear and beginner-friendly.
- Use numbered steps where possible.
- Include verification commands after configuration steps.
- Use relative image paths so GitHub renders screenshots correctly.
- Do not include real passwords, personal data, client data or internal business information.
- Use generic lab names such as `corp.local`, `SRV-DC01`, `W11-CLIENT01`.

## Screenshot Guidelines

Store screenshots under:

```text
assets/images/<lab-folder>/
```

Use lowercase file names with hyphens.

Example:

```text
01-windows-about-page.png
02-ipconfig-output.png
03-domain-join-success.png
```
