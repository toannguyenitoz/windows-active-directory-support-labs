# Contributing

This repository is designed as a practical step-by-step guide for Windows, Active Directory and IT Support practice.

## Contribution Guidelines

When adding or updating content:

- Keep instructions clear and beginner-friendly.
- Use numbered steps where possible.
- Include verification commands after configuration steps.
- Use relative image paths so GitHub can render screenshots correctly.
- Do not include passwords, personal data, client data or internal business information.
- Use generic lab names, for example `corp.local`, `SRV-DC01`, `W11-CLIENT01`.

## Screenshot Guidelines

Store screenshots under:

```text
assets/images/<lab-folder>/
```

Use lowercase file names with hyphens, for example:

```text
01-windows-about-page.png
02-ipconfig-output.png
03-domain-join-success.png
```

## Style

Use this structure for each lab:

```text
Overview
Objectives
Lab Environment
Step-by-step Instructions
Verification
Screenshot List
Completion Checklist
Key Takeaways
```
