# Lab 01 - Windows 11 Client Initial Configuration

This lab prepares a Windows 11 client for later Windows Server and Active Directory labs.

## Steps

1. Open Settings > System > About.
2. Rename the computer to W11-CLIENT01.
3. Restart the computer.
4. Check the computer name.
5. Check network settings.
6. Review power and Remote Desktop settings.
