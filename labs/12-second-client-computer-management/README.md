# Lab 12 - Second Client and Computer Management

## Overview

This lab adds a second Windows 11 client to the environment.

## Steps

1. Start the second Windows 11 client.
2. Rename it to W11-CLIENT02.
3. Restart the client.
4. Review network settings.
5. Join it to the lab domain.
6. Confirm the computer object appears in Active Directory.
7. Move the computer object to the correct OU.

## Checklist

- [ ] Second client renamed
- [ ] Network settings reviewed
- [ ] Domain join completed
- [ ] Computer object organized
