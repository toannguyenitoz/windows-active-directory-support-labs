# Screenshot Guide

Screenshots are used to make each lab easier to follow.

## Screenshot Folder Pattern

Each lab has its own image folder:

```text
assets/images/lab-xx-topic-name/
```

Example:

```text
assets/images/lab-01-windows-11-client-initial-configuration/
```

## File Naming Pattern

Use lowercase file names and hyphens:

```text
01-windows-about-page.png
02-rename-computer.png
03-command-output.png
```

## Recommended Screenshot Tool

Use **Snipping Tool**:

```text
Start > Snipping Tool > New
```

Save screenshots directly into the correct lab image folder.

## What to Capture

Capture only what helps the reader follow the guide:

- Settings pages
- Command outputs
- Wizard steps
- Successful configuration results
- Error messages during troubleshooting

Avoid exposing personal data, usernames, email addresses, serial numbers or private network details.
